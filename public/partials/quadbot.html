12 dof Quadruped Robot

Overview

The scope of this project was to build a quadrupedal robot controllable realtime through a web interface or a custom bluetooth remote controller. The project realized my senior year of high school.

Hardware

The robot is made of 3d prints designed on AutoCAD. The frame hold 12 Servo motors which are powered through an external power supply and regulated with a Buck converter. The servos are driven by Adafruitâ€™s 16-Channel PWM / Servo HAT.

Web Interface

The robot hosts a Flask server with sockets to provides client - server communication. User input is collected through game controllers.
