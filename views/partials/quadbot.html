12 dof Quadbot

Overview

The scope of this project was to build a quadrupedal robot controllable through a web interface or a custom bluetooth remote controller. The project was started my senior year of high school and finished midway through my freshman year of college.

Supporting code can be found at my github

Hardware

The robot is made of 3d prints designed using AutoCAD for Mac 2016. The frame hold 12 Standard Servo motors which are powered through an external power supply which is regulated by the DROK® LM2596 Digital Control Voltage Regulator DC Buck Converter. The servos are driven by Adafruit’s 16-Channel PWM / Servo HAT . Software runs off a Raspberry Pi

Web Interface

The robot runs a Flask server with socket.io python plugin which provides client - server communication over the websocket protocol. The front end is presented with bootstrap, jQuery and noioslider. User input is collected with nouislider.js, and uses the web based GamepadAPI for ps3 and xbox controller input

Robot Software

The robot’s software is loosely designed around a micro service architecture, in which independently deployable, small, modular services are run in which each service contains a unique process and communicates through a well-defined, lightweight mechanism to serve a distinct goal. In practice each ‘service’ is not modular and there is not a separate message system for each service.

Services:

  1. I2C Driver
  2. LaserDriver
  3. MotionCalculator
  4. MotorDriver
  5. PWMDriver
  6. Utilities
